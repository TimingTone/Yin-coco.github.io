Binary Tree
Created 2019-10-04| Updated 2019-10-04|DSTree
Word count: 234|Reading time: 1 min|Post View:

Vector and List belong to linear data structures,
but Tree isn’t one kind of data structure in the strict sense.

Tree

Rooted Tree

The Tree structure is nothing but a kind of special graph. That is to say, it can be regarded as a set of objects where some pairs of objects are connected by lines. Some of this objects may have this relation, then we introduce an edge. Then all the objects that can be defined relation between them will be called vertex.

Vertex is different from the Node we use in computer programming.

The Tree we assign here has following feature:

  • We will assign a special Node as root. This Tree will be called Rooted Tree.

  • By connecting each other, a small rooter tree that can be a larger rooted tree step by step.

  • Compared to this larger tree, every small tree of its parts is called subtree. These subtrees will have a new root r.

Ordered Tree

Under the foregoing conditions, the root of a subtree is called a child. On the other hand, the new root r is the father of these nodes. For any node r_i, the number of its children is called its degree.

In every single tree, the number of all edges equals to the whole degrees of all the nodes. At the same time, it equals the number of nodes plus 1.
